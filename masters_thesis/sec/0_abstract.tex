\begin{abstract}
To perceive, humans use memory to fill gaps caused by limited visibility, such as occlusion or restricted field-of-view. However, most 3D object detectors are limited to using sensor evidence from a short temporal window (0.1s-0.3s). In this work, we present a simple add-on for enhancing any existing 3D object detector with long-term memory regardless of its sensor modality and network architecture. We propose a model to fuse object proposals from a detector with object proposals from a memory bank of past predictions, exploiting trajectory forecasts to align objects across time. We propose a training schedule that balances data diversity and the gap between training and inference. By applying our method to existing LiDAR and camera-based detectors on the Waymo Open Dataset (WOD) and Argoverse 2 Sensor (AV2) dataset, we demonstrate significant improvements in performance, and achieve the top performance on the WOD 3D detection leaderboard among online methods.
\end{abstract}
